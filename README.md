# 🧠 CortexAI

CortexAI is a full-stack AI SaaS application similar to ChatGPT that allows users to generate AI responses and images using prompts.  
It includes authentication, chat history, image generation, community image sharing, and a credit-based payment system using Stripe.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 💬 AI Chat Interface
- 🖼️ AI Image Generation
- 📂 Chat History
- 🌍 Community Image Gallery
- 💳 Credit Based System
- 💰 Stripe Payment Integration
- ⚡ Fast Image Delivery via ImageKit
- 🎨 Dark / Light Theme Support
- 📱 Responsive UI
